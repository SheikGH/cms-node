Customer Management System – User Guide

I.	Frontend : React.js, Tailwind CSS with JWT Authentication
II.	Backend : Dot Net Core, Web API, Entity Framework Core, SQL Server with Clear Architecture

I. Frontend – React.js, Tailwind CSS with JWT Authentication

React Web Site Url: http://localhost:3000/

1. Register Page: To Add new customer
i. Check Form Validation: Validation Error message is showing if not enter proper customer details.
ii. Enter Customer valid details: Click 'Register' button then customer data go to backend api and save database

2. Login Page:

i. Check Form Validation:
Validation Error message is showing. 
If not enter the proper email and password.
If enter invalid email and password.

ii. Enter valid credential:
Email: abdul.rahman@gmail.com
Password: Abdul@123

Now Login credential email and password it send to backend api, api will validate email and password.
If credential is not valid return error message.
If credential is valid then create JWT token and retuen to the client.
React will save it in localStorage then redirect to Customer List page

3. Customer List Page: 
i. View Customer: 
Now we can see registed customer details in Customer List page

ii. Edit Customer: 
i. Check Form Validation: Validation Error message is showing if not enter proper customer details.
ii. Enter Customer valid details: Click 'Update' button then customer data go to backend api and save database then reload customer list page now display newly edited details.

iii. Delete Customer:
if click 'Delete' button on the Customer List page then delete customer record in the List.

4. Logout:
Once click 'Logout' button. 
Saved JWT token will delete.
Page will redirect to Login page
